Bem-vindo à Disciplina de PSW (Projeto de Sistemas Web)!

A disciplina de PSW (Projeto de Sistemas Web) oferece uma experiência prática completa no desenvolvimento de aplicações web. Durante o curso, você vai criar uma aplicação do zero, aprendendo a montar interfaces interativas com HTML, CSS e JavaScript e a usar frameworks como React e Node.js para adicionar funcionalidades avançadas.

Além do desenvolvimento front-end e back-end, você aprenderá a integrar bancos de dados (como MySQL e MongoDB) para armazenamento de informações e aplicar boas práticas de segurança para proteger sua aplicação. A disciplina também traz a experiência de trabalho em equipe, usando metodologias ágeis como Scrum, simulando o ambiente profissional.

Ao final, você aprenderá a publicar e manter a aplicação em plataformas como AWS e Heroku, concluindo o curso com um projeto completo e pronto para o portfólio.
